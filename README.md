# Coding Test

Aplikasi ini merupakan sistem sederhana untuk mengelola pengajuan kredit. Dibangun menggunakan teknologi Front End untuk memberikan pengalaman user yang cepat dan responsif

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Sonner (Toast Notification)
- React Icons
- Local Storage

## Fitur Utama

- Tambah data pengajuan kredit
- Validasi:
  - Pendapatan < 1 juta → ditolak
  - Nominal > 200 juta → ditolak
  - Maksimal 3 pengajuan per nasabah
- Approve / Reject pengajuan
- Hapus data pengajuan
- Statistik data pengajuan
- Notifikasi interaktif (toast)
